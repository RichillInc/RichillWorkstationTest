

# RichillWorkstation

## 一. 項目說明
RichillWorkstation是集合ERP(Enterprise Resource Planning)系統與ATS(Algorithmic Trading System)系統為一體的軟體

## 二. 項目資料夾結構

|DirectoryName|Description|
|:--:|:--|
|Devs|開發中程式存放區|
|Resources|資源檔案資料夾|
|Sources|原始碼存放資料夾|
|Tests|程式測試資料夾|
|Temps|未開發完成或不確定的程式存放的資料夾|
|Tools|開發工具程式資料夾|
